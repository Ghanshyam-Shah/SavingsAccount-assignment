# Savings Accounts Mobile App

This project is a React Native mobile application that displays savings account details fetched from a REST API.

## Features

- REST API integration (GET request)
- Loading indicator while fetching data
- Table-style mobile UI
- Circular navigation between accounts
- Splash screen
- Supports both Android and iOS

## API Used

https://srest.com/accounts.json

## How to Run

1. Clone the repository
2. Run `npm install`
3. For iOS: `npx pod-install`
4. Run the app:
   - Android: `npx react-native run-android`
   - iOS: `npx react-native run-ios`
